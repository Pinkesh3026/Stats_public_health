## About This Repository
This repository documents my personal journey learning statistical modelling, linear regression, and logistic regression using publicly available example datasets.
 
I am still learning, and I welcome any feedback, corrections, or suggestions along the way.
 
My broader research interest is in using large, complex longitudinal and cross-sectional datasets to understand relationships between predictor and outcome variables, particularly in biomedical and life science contexts.
 
---
 
## Learning Objectives
 
By working through these analyses, I aim to:
 
- Understand the assumptions underlying linear and logistic regression models
- Practise exploratory data analysis and variable selection
- Interpret model outputs including coefficients, confidence intervals, and p-values
- Visualise relationships between predictor and outcome variables
- Build a working foundation in R for future biomedical research applications

## How to Reproduce
 
1. Clone this repository:
   ```bash
   git clone https://github.com/pinkesh3026/Stats_public_health.git
   ```
 
2. Open the relevant script in RStudio.
3. Install required packages if not already available:
   ```r
   install.packages(c("tidyverse", "ggplot2", "broom"))
   ```
 
4. Run the script from top to bottom.
